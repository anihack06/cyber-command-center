# Cyber Center

Cyber Command Center is an interactive **cybersecurity dashboard simulation** designed to recreate a hacker-style command interface inside a web browser.

The project demonstrates a futuristic **cyber operations control panel** with a terminal command system, system monitoring tools, attack simulation, and global cyber activity visualization.

This interface is inspired by **cyberpunk and Matrix-style security dashboards**, built entirely using web technologies.

## Features

### Hacker-Style Terminal

A command-based terminal interface where users can interact with the system using simulated cybersecurity commands.

Commands available:

```
help
scan
ip
ai
db
hack
clear
```

The terminal displays system responses and simulated network operations.

---

### System Monitoring Dashboard

Real-time animated charts showing simulated:

* CPU usage
* Network traffic

Charts automatically update to create a live monitoring effect.

---

### Cyber Attack Simulation

A visual **attack simulation engine** that shows progress of a simulated cyber attack with dynamic system messages.

Example system messages:

```
Launching cyber attack...
Bypassing firewall...
Decrypting secure data...
Access granted
```

---

### Database Explorer

A simulated secure file database where users can browse files such as:

```
admin_keys.json
password_hashes.txt
encrypted_logs.bin
vpn_connections.cfg
crypto_wallet.dat
```

Clicking files simulates opening protected system data.

---

### Global Cyber Activity Map

A real-time activity feed displaying simulated cyber signals from various countries around the world.

Example:

```
[12:33:45] Signal detected from Russia
[12:33:48] Signal detected from India
```

---

### Matrix Background Interface

The interface includes a **Matrix-style animated background** to create a cyberpunk environment.
